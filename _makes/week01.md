## Q and A

1. What Is Being Made?
What is the output of this project?
The output of this project is an image of a fake person generated from a collection of real pictures.


What experience does it create for the user?
It is entertaining and interesting to see these pictures that look like real people while knowing that in actuality, they were created by a computer.



2. What Is the Project Made From? (Data)
"Data" for this project would consist of a large set of photos of real faces that the project uses to learn.


This project pulled around 70,000 images of faces from the website Flickr.


There is no interface for the actual website. The user gets one face when loading the website and then the only way to get another is to refresh the page.



3. Tools, Algorithms, or Systems
What software, platforms, or models are involved?
They use the StyleGAN (Style Generative Adversarial Network)to generate the faces.
Is AI used? If yes, where in the process?
Yes, generative AI is used to create realistic but fake faces.
What steps feel automated vs. guided?
There are no steps other than refreshing the page and no instructions or any other semblance of a guide.

4. Human Labor & Decisions
Who likely collected the data?
The data is collected by the website Flickr which is a popular place for photographers to post their images. The photos with faces are used to train TPDNE's GAN.
Who cleaned, categorized, or interpreted it?
TPDNE uses a tool called a GAN (Generative Adversarial Network) that first creates an image of a face then compares it against pictures of real faces until they are indistinguishable.
Where do you see human judgment shaping the outcome?
Humans are responsible for giving the initial database that is used to compare the generated faces for.

5. Design as Argument
How do visual or interface choices shape meaning?
Since there is no interface all the emphasis is put into the image that is generated by the website.
What does the project make easy to see?
The project is clearly trying to show off how realistic these synthetic faces can get.
What does it make hard to see?
This project is trying to make the line between humans and AI hard to see. It is very hard to look at the faces that are generated by this website and tell that it isn't a real person.
